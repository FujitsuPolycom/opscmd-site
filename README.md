# OpsCMD Website

Super simple static marketing page for the OpsCMD product family:

- OpsCMD Core
- OpsCMD Memory
- OpsCMD Roundtable
- optional OpsCMD Central
- gateway tools and earlier SysAudit work

Open `index.html` directly in a browser. No build step is required.

The contact form is browser-side only. Wire it to a CRM, Stripe flow, or onboarding API when the product flow is ready.
